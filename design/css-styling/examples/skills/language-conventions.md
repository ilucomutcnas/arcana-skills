# Examples — Naming & Language Conventions

## Good

```css
.article-card {}
.article-card__title {}
.article-card__meta {}
.article-card--featured {}
```

## Bad

```css
.statya-kartochka {}
.blue-title-final {}
.knopka_new2 {}
```

## Good comment

```css
/* Safari clips the focus ring here unless the overflow stays visible. */
```

## Bad comment

```css
/* This sets the color to blue. */
color: blue;
```
