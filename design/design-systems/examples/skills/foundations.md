# Foundations Example

## Task
Create a foundation model for a B2B SaaS admin product.

## Recommended output shape
- Primitive palette
- Semantic aliases: surface, text, border, accent, success, warning, destructive
- Typography roles: display, heading, title, body, label, caption
- Spacing scale: 4-based or 8-based, documented
- Radius scale: none, sm, md, lg, xl, pill
- Elevation scale: subtle, raised, overlay
- Breakpoints: content-driven, not device-marketing-driven

## Notes
Do not start from button colors. Start from the semantic system.
