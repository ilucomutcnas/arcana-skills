# Canvas Design — Examples

## Two-Step Process

1. **Design Philosophy Creation** — .md file explaining the aesthetic movement
2. **Visual Expression** — .pdf or .png file created on canvas

## Refinement Philosophy

When refining the visual output:
- Do not add more graphics — refine what exists.
- Ask: "How can I make what is already here more of a piece of art?"
- Respect the design philosophy and minimalism principles.
- Take a second pass to polish into a masterpiece.
