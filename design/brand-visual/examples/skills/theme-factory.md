## Workflow

1. Show `theme-factory__theme-showcase.pdf` to the user.
2. Ask which theme to apply.
3. Wait for explicit confirmation.
4. Read the selected theme file from `theme-factory__themes/`.
5. Apply the theme's colors and fonts to the artifact.

## Theme File Structure

Each theme file in `theme-factory__themes/` contains:
- Color palette with hex codes and usage notes
- Typography: header and body font selections
- Best-use context description
