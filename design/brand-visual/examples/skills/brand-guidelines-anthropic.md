# Brand Guidelines Anthropic — Examples

## Font System

- **Headings**: Poppins (fallback: Arial)
- **Body Text**: Lora (fallback: Georgia)

No font installation required — system fonts provide automatic fallback.

## Usage

Apply Anthropic brand identity when creating:
- Official documents and presentations
- Branded web artifacts
- Marketing materials
- Internal communications with brand standards
