# Design Orchestration — Examples

## Workflow Phases

1. **Ideation** → Define requirements, constraints
2. **Creation** → Route to canvas-design, design-md, or theme-factory
3. **Review** → Validate against brand-guidelines and web-design-codex
4. **Approval** → Confirm all criteria met
5. **Handoff** → Deliver validated design to implementation

## Key Principle

This skill does not generate designs. It controls the flow between other skills.
