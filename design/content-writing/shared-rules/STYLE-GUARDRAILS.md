# Shared Style Guardrails

Apply these rules across all writing skills unless the specific skill overrides them.

## Core Principles
- Write for the reader's goal, not for decorative performance.
- Prefer precision over flourish.
- Prefer structure over improvisation.
- Cut filler, throat-clearing, and empty intensifiers.
- Match tone to context, stakes, and audience.

## Never Do This
- Blend incompatible styles by accident.
- Sound AI-generic, inflated, or mechanically enthusiastic.
- State assumptions as facts.
- Invent evidence, names, numbers, dates, or legal effects.
- Overuse exclamation marks, rhetorical questions, or cliché transitions.
- Pad a text with repeated meaning.

## Internal Checks
Before finalizing, verify:
1. The task type was classified correctly.
2. The tone matches the context.
3. The structure matches the medium.
4. Unsupported claims were removed or softened.
5. Repetition was eliminated.
6. The opening sentence earns attention without sounding forced.
7. The ending stops cleanly instead of fading out.

## Formatting Defaults
- Plain prose unless the user asks for bullets, tables, clauses, or templates.
- Keep paragraphs purposeful.
- Use headings only when helpful.
- Preserve requested names, numbers, and terminology exactly unless correction is required.

## Escalation Rule
If the request carries legal, reputational, commercial, or public-information risk, slow down and become more explicit, source-aware, and conservative.
